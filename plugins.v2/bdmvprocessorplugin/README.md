# BDMVProcessor - 蓝光原盘自动重封装插件

MoviePilot v2 插件：自动扫描媒体库中的蓝光原盘 (BDMV) 目录，通过 Docker 容器中的 MakeMKV 重封装为 MKV 格式。

## 依赖

- MoviePilot 宿主容器内必须已安装 MakeMKV
- `makemkvcon` 命令全局可用

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 开关 | 关闭 |
| 定时运行策略 | Cron 表达式 | `0 3 * * *`（每天凌晨 3 点） |
| 媒体库根目录 | 多个目录用英文逗号分隔；留空则扫描全部本地媒体库 | — |
| 源文件清理策略 | 处理后是否删除原盘结构 | 不删除 |

## 工作流程

1. 按 Cron 表达式定时启动扫描任务
2. 递归遍历媒体库目录，查找包含 `BDMV` 子目录的电影目录
3. 跳过已存在大于 10GB MKV 文件的目录（视为已处理）
4. 调用 MakeMKV 自动识别最长正片并重封装为 MKV
5. 将生成的 MKV 移动至电影目录，清理临时文件
6. 按配置决定是否删除原盘源文件夹（`BDMV` / `CERTIFICATE`）
